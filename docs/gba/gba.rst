gba/gba.hpp
===========

Including the ``<gba/gba.hpp>`` header provides access to the gba-plusplus library through the ``gba`` namespace.

.. code-block:: cpp

    #include <gba/gba.hpp>

    int main() {
        using namespace gba;
        // ...
    }
